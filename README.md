# Chatbot AI avec DeepSeek

Une application web de chatbot utilisant l'API DeepSeek pour générer des réponses intelligentes.

## Installation

1. Installer les dépendances :
```bash
npm install
```

2. Configurer les variables d'environnement :
- Renommer le fichier `.env.example` en `.env`
- Mettre à jour la clé API DeepSeek dans le fichier `.env`

## Démarrage

1. Lancer le serveur :
```bash
npm start
```

2. Ouvrir `http://localhost:3000` dans votre navigateur

## Fonctionnalités

- Interface utilisateur intuitive
- Réponses en temps réel
- Indicateur de frappe pendant le chargement
- Design responsive

## Technologies utilisées

- Frontend : HTML, CSS, JavaScript
- Backend : Node.js, Express
- API : DeepSeek Chat API
